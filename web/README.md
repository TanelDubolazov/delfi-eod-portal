### Image Alt Text

Set lead image alt text per post in frontmatter:

```md
---
heroImage: ./media/lead.jpg
heroImageAlt: Emergency teams inspecting storm damage near the harbor
---
```

Set alt text for additional images directly in markdown content:

```md
![Rescue workers clearing debris](./media/debris-cleanup.jpg)
![Floodwater across main road](./media/flooded-road.jpg)
```
